# 中国传统宗教造像艺术科普网站

## 📖 项目简介

这是一个专注于中国传统宗教造像艺术的科普网站，旨在向公众普及佛教和道教造像的历史演变、艺术特色和文化内涵。网站结合了学术研究成果与现代网页技术，为用户提供沉浸式的文化体验。

## 🎯 核心功能

### 1. 宗教造像历史科普
- **佛教造像演变**：从汉代传入到明清发展的完整历史脉络
- **道教造像艺术**：从魏晋早期到明清集大成的发展历程
- **时间线展示**：直观的历史发展时间轴，配有详细的时代特征说明

### 2. 知名寺庙道观数据库
收录了中国前20个最具代表性的寺庙道观，包括：

#### 佛教寺庙（10个）
- **白马寺**（河南洛阳）- 中国第一座官办寺院
- **少林寺**（河南登封）- 禅宗祖庭
- **灵隐寺**（浙江杭州）- 江南著名古刹
- **天台山国清寺**（浙江台州）- 天台宗祖庭
- **九华山化城寺**（安徽池州）- 地藏菩萨道场
- **普陀山普济寺**（浙江舟山）- 观音菩萨道场
- **峨眉山万年寺**（四川乐山）- 普贤菩萨道场
- **五台山显通寺**（山西忻州）- 文殊菩萨道场
- **雍和宫**（北京）- 藏传佛教寺院
- **玉佛禅寺**（上海）- 近代都市佛教代表

#### 道教宫观（10个）
- **白云观**（北京）- 全真道祖庭
- **青城山天师洞**（四川成都）- 天师道祖庭
- **武当山紫霄宫**（湖北十堰）- 武当山主宫
- **崂山太清宫**（山东青岛）- 全真道第二丛林
- **龙虎山天师府**（江西鹰潭）- 正一道祖庭
- **楼观台**（陕西西安）- 道教发源地
- **华山西岳庙**（陕西渭南）- 西岳大帝祭祀场所
- **南岳衡山南岳大庙**（湖南衡阳）- 南岳大帝祭祀场所
- **中岳庙**（河南登封）- 中岳大帝祭祀场所
- **北岳庙**（河北保定）- 北岳大帝祭祀场所

### 3. 智能搜索与筛选
- **地理位置筛选**：按省份和城市进行精确搜索
- **宗教类型筛选**：区分佛教寺庙和道教宫观
- **实时搜索**：动态更新搜索结果

### 4. 造像图片智能分析
- **图片上传**：支持拖拽和点击上传，最大10MB
- **智能识别**：基于学术研究的造像特征分析
- **风格判断**：识别造像的艺术风格和年代特征
- **地域分析**：判断造像的地域文化特色
- **学术依据**：提供权威的学术研究支撑

### 5. 学术研究模块
- **艺术发展史**：汉魏、隋唐、宋元、明清四个时期的详细分析
- **地域风格研究**：中原、江南、西南、华北四大地域风格对比
- **象征意义研究**：手势、服饰、法器的宗教象征含义
- **权威学术支撑**：基于宿白、金维诺、阮荣春等权威学者研究成果

## 🚀 使用指南

### 快速开始
1. **打开网站**：在浏览器中访问index.html
2. **浏览首页**：了解网站整体结构和功能
3. **探索造像历史**：查看佛教和道教造像的发展历程
4. **搜索寺庙**：使用筛选功能查找感兴趣的寺庙道观
5. **查看详情**：点击寺庙卡片查看详细信息
6. **上传分析**：上传神像图片进行智能分析

### 功能详解

#### 寺庙搜索
1. 选择省份（可选）
2. 选择城市（可选）
3. 选择宗教类型（可选）
4. 点击"搜索"按钮
5. 浏览搜索结果，点击查看详情

#### 图片分析
1. 点击或拖拽图片到上传区域
2. 等待图片上传完成
3. 查看智能分析结果
4. 了解造像的神祇、风格、年代等信息

#### 学术研究
1. 点击"📚 学术研究"按钮
2. 浏览艺术发展史
3. 了解地域风格差异
4. 学习象征意义知识
