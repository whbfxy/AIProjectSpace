# 中国传统宗教造像艺术科普网站

## 🏛️ 项目简介

这是一个关于中国传统宗教造像艺术的科普网站，专注于佛教和道教造像的演变历史知识普及。网站采用简约设计风格，配色与整体主题保持一致，提供丰富的教育内容和互动功能。

## ✨ 主要功能

### 1. 造像历史科普
- **佛教造像演变**：从汉代传入到明清发展的完整时间线
- **道教造像发展**：从魏晋早期到明清集大成的历史脉络
- **详细的历史时期介绍和艺术特征分析**

### 2. 寺庙道观查询
- **前20个知名寺庙道观**的详细信息
- **多维度筛选**：
  - 按省份筛选
  - 按城市筛选  
  - 按类型筛选（佛教/道教）
- **各殿供奉神祇信息**
- **造像演变主体介绍**

### 3. 智能造像分析
- **图片上传功能**：支持JPG、PNG格式，最大10MB
- **AI分析结果**：
  - 神祇识别
  - 造像风格分析
  - 年代特征判断
  - 地域特征识别
  - 艺术特点总结

### 4. 响应式设计
- 适配桌面、平板、手机等多种设备
- 流畅的用户体验和导航

## 🎨 设计特色

- **简约风格**：清爽的界面设计，突出内容本身
- **传统配色**：采用符合传统文化主题的配色方案
- **优雅字体**：使用思源宋体，体现文化底蕴
- **流畅动画**：精心设计的过渡效果和微交互

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

**macOS/Linux用户：**
```bash
./start-server.sh
```

**Windows用户：**
```cmd
start-server.bat
```

### 方法二：手动启动

1. 确保已安装Python（推荐Python 3.6+）
2. 在项目目录下运行：
   ```bash
   python3 -m http.server 8000
   ```
3. 打开浏览器访问：http://localhost:8000

## 📱 功能使用指南

### 浏览造像历史
1. 点击导航栏"佛教造像"或"道教造像"
2. 滚动时间线查看不同历史时期
3. 点击各时期卡片查看详细介绍

### 查询寺庙道观
1. 导航到"寺庙道观"页面
2. 使用筛选器选择省份、城市和类型
3. 点击"搜索"按钮查看结果
4. 点击寺庙卡片查看详细信息

### 分析造像图片
1. 导航到"造像分析"页面
2. 点击或拖拽图片到上传区域
3. 等待分析完成
4. 查看详细的分析结果
