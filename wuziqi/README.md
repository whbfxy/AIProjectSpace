# 五子棋游戏

一个使用HTML5 Canvas开发的网页版五子棋游戏。

## 功能特性

- 🎮 **完整的游戏逻辑**：标准的五子棋规则，黑白双方轮流下棋
- 🤖 **人机对战**：支持与AI机器人对战，三种难度可选
- 🏆 **智能胜负判断**：自动检测横、竖、斜方向的五子连珠
- 🔄 **游戏控制**：支持重新开始和悔棋功能
- 🎨 **精美界面**：现代化的UI设计，渐变色彩和动画效果
- 📱 **响应式设计**：适配不同屏幕尺寸
- ⭐ **视觉效果**：棋子3D渐变效果、阴影、获胜连线高亮

## 游戏规则

1. 黑棋先行，白棋后手
2. 双方轮流在棋盘交叉点上放置棋子
3. 先在横、竖、斜方向上连成五子的一方获胜
4. 棋盘填满无人获胜则为平局

## 操作说明

- **下棋**：点击棋盘空白交叉点放置棋子
- **重新开始**：点击"重新开始"按钮重置游戏
- **悔棋**：点击"悔棋"按钮撤销上一步操作
- **切换游戏模式**：点击"双人对战"或"人机对战"按钮切换游戏模式
- **选择AI难度**：在人机对战模式下，可选择简单、中等、困难三种难度

## 游戏模式

### 双人对战
- 两位玩家轮流下棋，黑棋先行
- 适合两人对战娱乐

### 人机对战
- 玩家执黑棋先行，AI执白棋后手
- **简单难度**：AI随机下棋，适合初学者
- **中等难度**：AI具备基本攻防意识，会阻止玩家获胜并寻找获胜机会
- **困难难度**：AI具备较强的棋形分析能力，会考虑棋形发展潜力

## 技术实现

- **前端技术**：HTML5、CSS3、JavaScript ES6+
- **绘图技术**：Canvas API绘制棋盘和棋子
- **设计模式**：面向对象编程，使用ES6 Class
- **算法**：深度优先搜索(DFS)进行胜负判断

## 核心算法

### 胜负判断算法

使用深度优先搜索(DFS)算法，从当前落子位置向四个方向（横、竖、左斜、右斜）检查连续相同颜色的棋子数量：

```javascript
checkWin(row, col) {
    const directions = [
        [[0, 1], [0, -1]],   // 横向
        [[1, 0], [-1, 0]],   // 竖向
        [[1, 1], [-1, -1]],  // 左斜
        [[1, -1], [-1, 1]]   // 右斜
    ];
    
    for (const direction of directions) {
        const line = this.getLine(row, col, direction, player);
        if (line.length >= 5) {
            return true;
        }
    }
    return false;
}
```
